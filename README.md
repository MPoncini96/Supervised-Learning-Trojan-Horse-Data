# Supervised Learning on Trojan Horse Dataset

This repository contains a supervised learning workflow for classifying the **Trojan Horse** dataset. The project explores multiple ML models, evaluates their performance, and analyzes which features most strongly affect predictions.

---

## Pipeline Overview

1. **Load & Clean Data**  
   - Import Trojan Horse dataset  
   - Handle missing values, encode categorical variables, normalize features  

2. **Train/Test Split**  
   - Create training and evaluation subsets  


3. **Model Training (Classifiers Used)**  
   - **Logistic Regression**  
   - **Decision Tree**  
   - **Random Forest**  
   - **k-Nearest Neighbors (k-NN)**  
   - **Support Vector Machine (SVM)**  
   - **Na√Øve Bayes**
   - **Gradient Boosting / XGBoost**
   - **Ensemble**

4. **Evaluation**  
   - Classification metrics  
   - Confusion matrices  
   - Feature importance (if applicable)

5. **Visualization**  
   - Plots of model performance and decision boundaries
